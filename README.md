# Doctor Model Video Toolkit

Design a stylized doctor character for motion graphics, virtual production, or explainer content. This Next.js experience lets you tweak the coat palette, accessories, and descriptive prompt while previewing a 3D doctor rendered live in the browser.

## Quickstart

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the model designer.

## Available Scripts

- `npm run dev` – start the development server
- `npm run build` – create a production build
- `npm start` – serve the production build
- `npm run lint` – run the Next.js ESLint rules

## Features

- Preset lookbook with cinematic doctor styles
- Interactive color controls for coat, scrubs, and skin tone
- Accessory toggles (stethoscope, clipboard, or minimal)
- Real-time 3D preview using React Three Fiber and Drei
- Auto-generated creative prompt for video direction

## Tech Stack

- Next.js App Router
- React 18 with TypeScript
- React Three Fiber & Drei
- Three.js
- CSS-based glassmorphism styling
